# Farm2Spice - Premium Spices Export Website

## Project Overview

Farm2Spice is a professional website redesign inspired by the PTEXIM layout structure, created for a Ghanaian spice export business. The website positions Farm2Spice as a premium international spice supplier while showcasing their sustainable farming and community impact mission.

### How to Deploy Updates

1. **GitHub Repository Method**:
   ```bash
   # Update your GitHub repository
   git add .
   git commit -m "Website improvements"
   git push origin main
   ```

2. **Vercel Automatic Deployment**:
   - Connected to GitHub repository
   - Automatically deploys on push to main branch
   - No manual intervention required

3. **Manual Deployment**:
   - Use files in `vercel-deployment/` folder
   - Upload directly to hosting provider
   - Ensure all paths are correctly configured
